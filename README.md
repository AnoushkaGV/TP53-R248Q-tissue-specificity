## Project Overview

This project investigates the **TP53 gene** variant **rs11540652** and its potential clinical implications.  

- **SNP (Variant):** rs11540652  
- **Allele Change:** C > A, G, T  
- **Protein Change (non-synonymous):** R > L, P, Q  
- **SNP Location:** chr17:7,674,220 (hg38)  
- **Region Analyzed:** chr17:7,674,181–7,674,290 (exon), full gene region: chr17:7,668,421–7,687,490  

## Clinical Relevance

The SNP affects **TP53** and is associated with:  
- Hereditary cancer-predisposing syndrome  
- Myelodysplastic syndrome  
- Li-Fraumeni syndrome  

## Data & Analysis

Data and scripts are included for:  
- Gene and exon visualization  
- SNP annotation  
- Survival analysis for TP53 R248 mutations across cancer types  

## Usage

Clone this repository and run the provided R scripts to reproduce the analyses and visualizations.
